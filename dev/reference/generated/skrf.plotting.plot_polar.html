skrf.plotting.plot_polar¶

skrf.plotting.plot_polar(theta, r, x_label=None, y_label=None, title=None, show_legend=True, axis_equal=False, ax=None, *args, **kwargs)¶

theta : array-like

data to plot

r : array-like

x_label : string

y_label : string

title : string

plot title
