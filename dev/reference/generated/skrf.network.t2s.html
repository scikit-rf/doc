skrf.network.t2s¶

skrf.network.t2s(t)¶

converts scattering transfer parameters [1] to scattering parameters [2]

transfer parameters are also refered to as ‘wave cascading matrix’, this function only operates on 2-port networks. this function only operates on 2-port scattering parameters.

Parameters :

t : numpy.ndarray (shape fx2x2)

scattering transfer parameters
Returns :

s : numpy.ndarray

scattering parameter matrix.

See also

calculates inverse s-parameters
