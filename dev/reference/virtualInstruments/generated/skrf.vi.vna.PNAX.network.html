skrf.vi.vna.PNAX.network¶

PNAX.network¶

Initiates a sweep and returns a Network type represting the data.

If you are taking multiple sweeps, and want the sweep timing to work, put the turn continuous mode off, with pnax.continuous='off'

Examples

>>> from skrf.vi.vna import PNAX 
>>> v = PNAX()
>>> dut = v.network
