skrf.network.Network.read¶

Network.read(*args, **kwargs)¶

Read a Network from a ‘ntwk’ file

Parameters :

*args, **kwargs : args and kwargs

passed to skrf.io.io.write()

See also

write(), skrf.io.io.write(), skrf.io.io.read()
